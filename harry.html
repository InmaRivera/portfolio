Página Web de harry potter

Aplicación web desarrollada con PhP, Node.js, JavaScript, jQuery, HTML, CSS, Bootstrap, ficheros JSON y la base de datos no relacional (NoSQL) MongoDB.
El objetivo del proyecto era gestionar una colección de personajes del universo de Harry Potter. La aplicación permite importar los personajes a la base de datos, añadir nuevos, eliminarlos o filtrarlos según distintos criterios mediante botones ubicados en la parte superior de la interfaz.
Al eliminar un personaje, este desaparece automáticamente y se muestra un mensaje de confirmación. De forma similar, al añadir un nuevo personaje, el sistema valida que los campos estén correctamente completados y notifica si falta algún dato o si la operación se realizó con éxito.
