Página Web de Reyes Magos

Aplicación web desarrollada con PHP, HTML, CSS y base de datos relacional MySQL.
El proyecto consiste en una plataforma pensada para que los Reyes Magos puedan gestionar la lista de niños. Desde la aplicación, pueden consultar, modificar o eliminar registros de los regalos niños registrados en el sistema.
Además, incluye varias secciones, como una lista de regalos, donde también es posible ver, editar o eliminar entradas. Las otras dos secciones están dedicadas a tablas de consulta que facilitan la visualización de información relacionada.
